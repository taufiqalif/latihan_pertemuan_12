# String


## Tetapkan String ke Variabel
Menetapkan **string** ke **variabel** dilakukan dengan nama variabel diikuti dengan tanda sama dengan dan string:

	var1 = 'hello world'
	var2 = "python programing"

	print(("var1[0]: "), var1[0])
	print(("var2[1:5]: "), var2[1:5])
